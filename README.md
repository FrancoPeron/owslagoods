## Installation

1. ´Clone the repo´
   ```sh
   git clone https://github.com/FrancoPeron/owslagoods.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Server App
   ```sh
   npm run dev
   ```
